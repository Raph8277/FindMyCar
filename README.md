> **70–85 ch · ≤ 5 000 € · ≤ 150 000 km**

## 🌐 Déploiement GitHub Pages

### 1. Créer le dépôt GitHub

```bash
git init
git add index.html README.md
git commit -m "FindMyCar - rapport interactif"
git branch -M main
git remote add origin https://github.com/VOTRE_PSEUDO/findmycar.git
git push -u origin main
```

### 2. Activer GitHub Pages

1. Aller sur le dépôt → **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : `main` · Folder : `/ (root)`
4. Cliquer **Save**

➡️ Votre site sera disponible sur : `https://VOTRE_PSEUDO.github.io/findmycar/`
